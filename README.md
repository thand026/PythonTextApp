# PythonTextApp
A semi-fullstack text application built using python.
This application is designed to showcase the programming logic and understanding of programming as a beginner. It was part of an assignment I had during my first year of software development. it makes use of basic fundamentals (loops, user input,output, etc) to make an application that can be used on a daily basis. 
Disclaimer: this application is not fully complete as it requires data storage and retreival which is something I am currently in the process of learning. After learning data storage and retreival, this application will be updated.

# How it Works
The way this program works is by registering a new user and the data is stored in memory. After registration, the user can log in, and decide to send a message to a recepient of their choosing. 
Disclaimer: this application will not send a message. It is only a demonstration.
The user can choose how many messages they would like to send and the program will respond accordingly, generating a message ID for each message sent. 
To run the program, launch python in the terminal and select the 'textApp.py' file:
  python textApp.py
